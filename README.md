# Supa CRM

## Built With

* NextJS
* TailwindCSS
* Supabase
* Stripe

## How To Use

Start the development server:
```
npm run dev
```

Build the app for production:
```
npm run build
```

Run the built app in production mode:
```
npm start
```

Switch to the currently recommended Node version:
```
nvm use
```
